# mongo-upload-images
Express JS App for uploading of images
## Démarrer la base de données Mongo DB sur Docker
`
docker-compose up -d
`

## Installer les dépendances
`
npm install
`

## Démarrer l'application
`
npm start
`

## Acceder à l'application
[Acceder à l'application](http://localhost:3000)
